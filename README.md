# NLP Chatbot Project

This is my final project for my internship at TECHTERN.
I built a simple chatbot using Python and NLP that can answer basic questions about Python, Machine Learning and AI.

---

## What this chatbot can do

- Answer questions about Python, Machine Learning, Deep Learning and NLP
- Answer questions about Data Science and Artificial Intelligence
- Handle unknown inputs and give a default message
- Have a basic conversation like hello, bye, thank you etc.

---

## How it works

1. User types a question
2. The chatbot preprocesses the input using tokenization, stopword removal and stemming
3. It converts the text to numbers using TF-IDF Vectorizer
4. It finds the most similar question from the knowledge base using Cosine Similarity
5. It returns the best matching answer

---

## Tools and Libraries used

- Python 3
- NLTK
- Scikit-learn
- NumPy
- Jupyter Notebook

---

## How to run

1. Open the notebook in Jupyter Notebook or Google Colab
2. Run all the cells one by one
3. Type your question in the input box and press Enter
